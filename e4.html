Projet E4

Hôpital Sud Paris (HSP)

Modernisation et sécurisation de l'infrastructure réseau

Contexte du projet

La Générale des hôpitaux (GDH)

Créée en 1984, GDH est le premier groupe de cliniques et hôpitaux privés en France. Avec 40 années d'expérience, le groupe a su intégrer les dernières technologies médicales pour offrir des prestations de qualité.

  • Siège social situé à Paris
  • Leader dans le secteur hospitalier privé
  • Expertise technologique avancée

Infrastructure réseau

État des lieux

  • Réseau de 350 postes utilisateurs
  • Utilisateurs variés (praticiens, administratifs, patients)

Enjeux

  • Sécurisation des données médicales
  • Haute disponibilité des services

Solutions mises en place

Architecture réseau sécurisée

DMZ et Proxy

  • Mise en place d'une DMZ pour les services exposés
  • Proxy sécurisé pour le filtrage du trafic Web

Services Web

  • Site web de prise de rendez-vous
  • Portail professionnel de santé
  • Système de notification automatisé

Infrastructure Active Directory

Serveurs AD

  • Contrôleur de domaine principal
  • Contrôleur secondaire pour redondance
  • Réplication automatique des données
  • Sauvegarde renforcé
  • BitLocker

Gestion des accès

  • Stratégies de groupe personnalisées
  • Gestion centralisée des utilisateurs

Messagerie Exchange

  • Serveur Exchange pour la messagerie professionnelle

Segmentation réseau

Mise en place de VLANs pour séparer les différents services :

  • VLAN VISIO
  • VLAN LABO
  • VLAN OPHTALMO
  • VLAN ADMINISTRATIF
  • VLAN ANGIO
  • VLAN PRATICIEN
  • VLAN PATIENT
  • VLAN VISITEUR
  • VLAN SERVEURS
  • VLAN DMZ
  • VLAN WAN

Sécurisation des accès

  • Déploiement d'un serveur RADIUS
  • Configuration des ACLs sur les switchs
  • Installation d'un pare-feu

Haute disponibilité

  • Redondance des équipements critiques
  • Configuration HSRP sur les routeurs
  • Mise en place d'une borne WIFI
  • Mise en place de GLPI

Compétences mobilisées

Techniques

  • Configuration des switchs
  • Routage avancé
  • Sécurité réseau
  • Monitoring

Organisationnelles

  • Gestion de projet
  • Planification
  • Documentation
  • Communication

Relationnelles

  • Travail en équipe
  • Relation client
  • Formation utilisateurs
  • Reporting
