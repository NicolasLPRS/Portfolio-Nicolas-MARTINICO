BTS SIO SISR

Services Informatiques aux Organisations

Option Solutions d'Infrastructure, Systèmes et Réseaux

La formation

Le BTS SIO (Services Informatiques aux Organisations) est une formation de niveau Bac+2 qui prépare aux métiers de l'informatique. L'option SISR (Solutions d'Infrastructure, Systèmes et Réseaux) forme des professionnels capables de gérer et d'administrer le réseau d'une organisation.

Cette formation est réalisée en alternance chez STELOGY, me permettant d'acquérir une expérience professionnelle concrète tout en poursuivant mes études.

Compétences acquises

Administration système

  • Configuration de serveurs Windows Server
  • Gestion d'Active Directory
  • Administration de systèmes Linux
  • Virtualisation (VMware, Hyper-V)

Réseaux

  • Configuration de switches et routeurs
  • Mise en place de VLANs
  • Sécurisation des réseaux
  • Protocoles de routage

Sécurité

  • Mise en place de pare-feu
  • Configuration de VPN
  • Gestion des droits d'accès
  • Sécurisation des données

Cloud Computing

  • Microsoft Azure
  • Services cloud Microsoft 365
  • Solutions de sauvegarde cloud
  • Gestion des identités cloud

Projets réalisés

Migration infrastructure

Migration d'une infrastructure physique vers une solution virtualisée, incluant la mise en place de serveurs virtuels et la configuration du réseau.

Déploiement Microsoft 365

Configuration et déploiement de Microsoft 365 pour une entreprise, incluant la migration des emails et la mise en place des politiques de sécurité.
